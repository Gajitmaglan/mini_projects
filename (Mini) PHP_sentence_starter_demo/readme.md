## How to run

Open these using server (example: if you use XAMPP, place project in htdocs and run the server).
Open index.html with the server.

## Purpose

Supposed as an English Sentence Starter Trainer.

## How to use

Translate as fast as you can from Russian to English, using the provided Sentence Starter.
Check answer and move on to the next line.
Speed is more important that accuracy, you should be able to use the provided sentence starter fluently.
